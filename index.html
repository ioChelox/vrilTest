Install App 3.0.0 (1)

Install App
If installation doesn't start, please check the requirements below.

Installation Requirements:

  • Use Safari browser on iOS device
  • Enable pop-ups in Safari
  • Trust developer certificate in Settings
  • Ensure device is not in Low Power Mode
  • Have stable internet connection
Having issues? Try these steps:
  1. Go to Settings > General > Device Management
  2. Trust the developer certificate
  3. Return to this page and try again
